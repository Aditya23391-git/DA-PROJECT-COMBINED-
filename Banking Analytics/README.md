# Banking Risk Analytics Dashboard

## Project Description

The Banking Risk Analytics Dashboard is designed to provide insights into customer financial behavior and lending risk within the banking and financial services sector. The project focuses on understanding how financial institutions leverage customer data to minimize the risk associated with lending operations and improve decision-making processes.

The dashboard integrates customer demographics, banking activities, deposits, loans, income patterns, and risk classifications to identify financially stable customers and evaluate overall portfolio risk exposure.

---

# Problem Statement

Develop a basic understanding of risk analytics in banking and financial services and understand how data is used to minimise the risk of losing money while lending to customers.

---

# Project Objectives

* Analyze customer financial profiles and banking behavior
* Identify high-risk and low-risk customer segments
* Evaluate the relationship between income, deposits, and loan exposure
* Understand how banking institutions reduce lending risk through analytics
* Develop an interactive and data-driven dashboard for financial analysis

---

# Technology Stack

* **SQL** – Data extraction, querying, and database management
* **Python** – Data cleaning, preprocessing, and analytical operations
* **Power BI** – Dashboard development and data visualization
* **CSV Files** – Data storage and transformation

---

# Project Workflow

## Data Extraction using SQL

Customer banking data was stored and queried using SQL to retrieve relevant financial and demographic information required for analysis.

## Data Processing using Python

The extracted data was processed in Python for:

* Data cleaning
* Handling missing values
* Feature preparation
* Data transformation and preprocessing

## Dashboard Development using Power BI

The processed dataset was connected to Power BI through SQL integration to create an interactive Banking Risk Analytics Dashboard featuring:

* KPI metrics
* Loan analysis
* Deposit analysis
* Risk categorization
* Customer segmentation
* Interactive filters and slicers

---

# Key Insights

* Medium-risk customers represent the largest segment of the customer base.
* High-risk customers demonstrate higher loan exposure levels.
* Customers with higher estimated income maintain stronger deposits and comparatively lower financial risk.
* Loyalty classifications such as Jade and Platinum contribute significantly to overall bank deposits.
* Property ownership positively influences business lending eligibility.
* Savings and checking accounts constitute the majority of customer financial holdings.
* Risk-based customer segmentation enables banks to reduce financial losses and improve lending efficiency.
