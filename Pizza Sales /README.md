# Pizza Sales Analysis

## Project Overview

This project focuses on analyzing pizza sales data using SQL to understand overall business performance, customer ordering patterns, and product-level trends. The goal was to use SQL queries to generate useful business insights such as revenue, order trends, category-wise performance, and best-selling pizzas.

The project helped me practice writing analytical SQL queries for KPI tracking, trend analysis, and sales breakdowns in a food sales dataset.

---

## Problem Statement

To analyze pizza sales data and extract meaningful business insights related to revenue, customer orders, product demand, and sales trends using SQL.

---

## Objectives

* Calculate key business KPIs such as revenue, total orders, and average order value
* Analyze daily and hourly order trends
* Study sales contribution by pizza category and pizza size
* Identify top-selling and lowest-selling pizzas
* Understand customer ordering behavior using SQL-based analysis

---

## Tools & Technologies

* **MySQL** – used for writing and executing SQL queries
* **SQL** – used for KPI analysis, trend analysis, and sales breakdown
* **CSV Dataset** – pizza sales transaction dataset used for analysis

---

## Dataset Description

The dataset contains transactional pizza sales data, including:

* `order_id`
* `order_date`
* `order_time`
* `pizza_name`
* `pizza_category`
* `pizza_size`
* `quantity`
* `total_price`

These fields were used to calculate sales KPIs, order trends, and product-level insights.

---

## Project Workflow

### 1. KPI Analysis

SQL queries were written to calculate important business metrics such as:

* Total Revenue
* Average Order Value
* Total Pizzas Sold
* Total Orders
* Average Pizzas Per Order

### 2. Trend Analysis

Order trends were analyzed to understand customer purchase behavior over time:

* Daily trend for total orders
* Hourly trend for total orders

### 3. Sales Breakdown Analysis

The dataset was further analyzed to understand how sales were distributed across different pizza categories and sizes:

* Percentage of sales by pizza category
* Percentage of sales by pizza size
* Total pizzas sold by category

### 4. Product Performance Analysis

SQL queries were used to identify the best-performing and worst-performing pizzas:

* Top 5 best sellers by total pizzas sold
* Bottom 5 sellers by total pizzas sold
